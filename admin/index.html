如何使用 LaobaiEPG

1. IPTV 播放器配置

在你的 IPTV 播放器（TiviMate、Kodi、Perfect Player 等）中设置 EPG 地址：

EPG URL: https://your-worker.workers.dev/guide.xml
       或
EPG URL: https://your-worker.workers.dev/guide.xml.gz（推荐，更小）

2. M3U 播放列表格式

你的 M3U 播放列表中的 tvg-name 会自动匹配到对应的频道节目单：

#EXTM3U
#EXTINF:-1 tvg-name="浙江卫视4K" group-title="卫视",浙江卫视
http://stream.example.com/zjtv.m3u8

#EXTINF:-1 tvg-name="CCTV1" group-title="央视",CCTV-1
http://stream.example.com/cctv1.m3u8

3. 添加新频道

编辑 GitHub 仓库中的 config/channels.yaml：

channels:
  - id: "MyChannel"
    name: "我的频道"
    group: "卫视"
    aliases:
      - "我的频道"
      - "MyChannelHD"
      - "我的频道4K"
    sources:
      - type: "xmltv_url"
        name: "我的频道"   # 在 XMLTV 源中的频道名

4. 添加数据源

编辑 config/sources.yaml 中的 xmltv_sources：

xmltv_sources:
  - id: "my_source"
    name: "我的数据源"
    url: "https://example.com/guide.xml.gz"
    enabled: true

5. 部署步骤（首次使用）

  1. Fork 本项目到你的 GitHub 账号
  2. 在 Cloudflare 创建 R2 Bucket（名称：epg-data）
  3. 在 Cloudflare 创建 KV Namespace，记录 ID
  4. 将 wrangler.jsonc 中的 KV ID 替换为真实 ID
  5. 在 GitHub Secrets 中配置：CLOUDFLARE_API_TOKEN、CLOUDFLARE_ACCOUNT_ID、KV_NAMESPACE_ID
  6. 推送代码，GitHub Actions 自动部署 Worker 并执行首次抓取
